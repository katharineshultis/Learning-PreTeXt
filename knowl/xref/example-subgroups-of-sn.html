Example 12.0.9.

Let \(m,n\in{\mathbb{N}}\) and assume that \(n\leq m\text{.}\) Then \(S_n\leq S_m\text{.}\) Recall that we used \(X_n=\{1,2,\ldots,n\}\) in defining the group \(S_n\text{.}\) Certainly this subgroup claim requires that we allow a function \(\sigma:X_n\to X_n\) to be thought of as a function \(\sigma: X_m\to X_m\) by mapping each element \(k\in X_m\setminus X_n\) to itself.
