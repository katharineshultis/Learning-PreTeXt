    Recall that with rings, the definition of an ideal was ideally suited to define quotient rings and have the resulting operations be well-defined. The notion of a normal subgroup is analogous in the sense that we want something ideally suited to define a quotient group and have the operation be well-defined.
    ðŸ”—
    Letâ€™s think backwards a bit, and work with a group written additively. If \(G\) is a group, and \(N\) is a "normal" subgroup of \(G\text{,}\) we might want to define the "cosets" of \(N\text{,}\) i.e. the elements of the quotient group \(G/N\text{,}\) to be of the form \([g]=g+N=\{g+n\mid n\in N\}\) with \(g\in G\text{.}\) Suppose further that we want the operation on these to be given by \([g]+[h]=[g+h]\) for \([g],[h]\in G/N\text{.}\) For this to be well-defined, weâ€™d need that for \([g]=[g']\) and \([h]=[h']\text{,}\) then \([g+h]=[g'+h']\text{,}\) i.e., weâ€™d need that \(-(g'+h')+(g+h)\in N\) whenever \(-g'+g\in N\) and \(-h'+h\in N\text{.}\) Because we might be more prone to assuming commutativity in additive notation, and we should be careful about not assuming things weâ€™re not given, letâ€™s switch now to multiplicative notation!
    ðŸ”—
    If \(G\) is a group written multiplicatively, with identity element \(e\text{,}\) and \(N\) is a "normal" subgroup of \(G\text{,}\) we have "cosets" of \(N\) being of the form \(\overline{g}=[g]=gN=\{gn\mid n\in N\}\text{.}\) For the operation \((gN)(hN)=(gh)N\) to be well-defined, we will need to take \(a,a',b,b'\in G\) such that \(a'N=aN\) and \(b'N=bN\) and need to show that \((a'b')N=(ab)N\text{,}\) i.e. that \((ab)^{-1}(a'b')\in N\text{.}\) Being a bit more specific, note that \(nN=eN\) for any \(n\in N\text{,}\) and fix some \(g\in G\) and \(n\in N\text{.}\) For the operation to be well-defined in this specific instance, weâ€™ll need that \((nN)(gN)=(eN)(gN)\text{,}\) i.e. we need that \((eg)^{-1}(ng)\in N\text{.}\) Recall from Exercise 11.C.i that \((ab)^{-1}=b^{-1}a^{-1}\) for any \(a,b\in G\text{.}\) Also, note that the identity element is always its own inverse. Now, note that for the operation to be well-defined, we need that \((eg)^{-1}(ng)=(g^{-1}e^{-1})(ng)=g^{-1}eng=g^{-1}ng\) is an element of \(N\text{.}\) Weâ€™re ready for the definition of a normal subgroup now, as weâ€™ll create that definition to make this work.

    Definition 13.0.1. Normal Subgroup.

    Let \((G,\star)\) be a group and \(N\) a subset of \(G\text{.}\) We say that \(N\) is a normal subgroup of \(G\), and denote this as \(N\lhd G\text{,}\) if \(N\) is a subgroup of \(G\) and for every \(g\in G\) and \(n\in N\text{,}\) we have that \(g^{-1}\star n\star g\in N\) as well.

    Example 13.0.3. Subgroups of "our" Favorite Abelian Group.

    Our favorite abelian group might be \({\mathbb{Z}}\text{,}\) and our favorite subgroup of this is probably \(n{\mathbb{Z}}\) where \(n\) is our favorite nonzero integer. As \({\mathbb{Z}}\) uses additive notation, for \(n\in{\mathbb{Z}}\) to be a normal subgroup of \({\mathbb{Z}}\text{,}\) weâ€™d need that \(-g+nh+g\in n{\mathbb{Z}}\) whenever \(nh\in n{\mathbb{Z}}\) and \(g\in{\mathbb{Z}}\text{.}\) This is "easily" seen to work, as the commutativity of the operation allows for us to rewrite this as \((-g+g)+nh=nh\in n{\mathbb{Z}}\text{.}\)

    Example 13.0.5. Normal Subgroup of \(S_3\).

    Our smallest non-abelian group is \(S_3\text{.}\) Consider the set \(A_3=\{(1),(1\;2\;3),(1\;3\;2)\}\subset S_3\text{.}\) We can see, although you should ask yourself how, from this smaller part of the group table for \(S_3\) that \(A_3\) is a subgroup of \(S_3\text{:}\)

    To check that it is normal, we need that \(g^{-1}ng\in A_3\) for every \(n\in A_3\) and every \(g\in S_3\text{.}\) Note that it is clear that \(g^{-1}ng\in A_3\) if \(g,n\in A_3\text{,}\) so we really only need to test this for \(g\in S_3\setminus A_3\text{.}\) You should do these computations to get more comfortable with cycle notation for elements of a permutation group.

    Let \(G\) be a group written multiplicatively and \(H\) a subgroup of \(G\text{.}\) Recall that the left cosets of \(H\) in \(G\) are the sets of the form \(gH=\{gh\mid h\in H\}\text{.}\) We can similarly define the right cosets of \(H\) in \(G\) to be the sets of the form \(Hg=\{hg\mid h\in H\}\text{.}\)

    Theorem 13.0.7. Normal Subgroup Equivalent Statements.

    Let \(G\) be a group and \(H\) a subgroup of \(G\text{.}\) The following are equivalent:
    1. The subgroup \(H\) is a normal subgroup of \(G\text{.}\)
      ðŸ”—
    2. For every \(g\in G\text{,}\) we have \(gH=Hg\text{.}\) Note that this is a claim about set equality.
      ðŸ”—
    3. For every \(g\in G\text{,}\) we have \(g^{-1}Hg=H\) where \(g^{-1}Hg\) denotes the set of all elements of the form \(g^{-1}hg\) with \(h\in H\text{.}\) Note that the claim \(g^{-1}Hg=H\) is also a claim about set equality.
      ðŸ”—
    ðŸ”—
    ðŸ”—
    The equivalence of the first two items above is typically phrased to say that "A subgroup \(H\) is normal if and only if its left and right cosets are the same."

    Definition 13.0.8. Index of a Subgroup.

    Let \(H\) be a subgroup of a finite group \(G\text{.}\) We call the quotient \(\dfrac{|G|}{|H|}\) the index of \(H\) in \(G\) and denote it by \([G:H]\text{.}\)

    Theorem 13.0.9. Index 2 Subgroups.

    Let \(G\) be a finite group. If \(H\) is a subgroup of \(G\) of index \(2\text{,}\) then \(H\) is a normal subgroup of \(G\text{.}\)

Of course, we need to assume that \(H\) is a subgroup of \(G\) and that \(H\) has index \(2\text{.}\) Our goal/task is to show that \(H\) is normal, which, by TheoremÂ 13.0.4, we can do by proving \(gH=Hg\) for every \(g\in G\text{.}\) So let \(g\in G\text{.}\) There are two possibilities:

    • if \(g\notin H\text{,}\) then \(gH\cap H=\emptyset\) and \(Hg\cap H=\emptyset\text{.}\) As \(H\) has index \(2\) in \(G\text{,}\) then \(H\) is half of the elements of \(G\) and the other half must be both \(Hg\) and \(gH\) so that \(gH=Hg\) as desired.
