    As is typical in algebra, one definition of a subgroup is that it is a SUB-GROUP. That is, it is a subset of a group that is itself a group. Hereâ€™s a more formal definition.
    ðŸ”—

    Definition 12.0.1. Subgroup.

    Let \((G,\star)\) be a group. A set \(H\) is a subgroup of \(G\) if:
    1. The set \(H\) is a (nonempty) subset of \(G\text{.}\)
      ðŸ”—
    2. If \(h,k\in H\text{,}\) then so is \(h\star k\text{.}\)
      ðŸ”—
    3. The set \(H\) together with the operation \(\star\) is a group. That is, it satisfies the axioms of a group: the operation is associative; the operation has an identity element, denoted \(e\text{;}\) and, every element has an inverse with respect to \(\star\text{.}\)
      ðŸ”—
    ðŸ”—
    ðŸ”—

    Note 12.0.2.

    As with subrings, the word nonempty is in parentheses because the emptyset is not a group, and so cannot be a subgroup of any group. The issue is that a group must contain at least one element: the identity element.
    ðŸ”—
    ðŸ”—

    Note 12.0.3.

    Some (possibly most or all) texts use \(H\leq G\) to denote that \(H\) is a subgroup of \(G\text{.}\) The notation \(H\lneq G\) then means that \(H\) is a proper subgroup of \(G\text{,}\) meaning that \(H\neq G\) and \(H\) is a subgroup of \(G\text{.}\)

    Example 12.0.4. Simplest, always in existence, subgroups.

    Let \(G\) be a group with identity element \(e\text{.}\) Then \(\{e\}\) and \(G\) are subgroups of \(G\text{.}\) Is this clear?

    Note 12.0.5.

    Note that as with subrings, another way to rephrase the second item in the definition of a subgroup is that we say the set \(H\) is closed under the operation \(\star\text{.}\)

    Theorem 12.0.6. Subgroup Test.

    Let \((G,\star)\) be a group and \(H\) a nonempty subset of \(G\text{.}\) For any \(g\in G\text{,}\) we let \(g^{-1}\) denote the inverse of \(g\) in \(G\text{.}\) The set \(H\) is a subgroup of \(G\) if and only if \(g\star h^{-1}\in H\) for all \(g,h\in H\text{.}\)
    ðŸ”—
    ðŸ”—

    Proof.

    \((\Longrightarrow)\) First, we assume \(H\) is a subgroup of \(G\) and we must show that \(g\ast h^{-1}\in H\) for all \(g,h\in H\text{.}\) So let \(g,h\in H\text{.}\) As \(H\) is a subgroup of \(G\text{,}\) it is a group, and so the inverse of \(h\) is an element of \(H\text{,}\) i.e. \(h^{-1}\in H\text{.}\) As \(H\) is a subgroup of \(G\text{,}\) it is closed under multiplication and so we have \(g\ast h^{-1}\in H\) as desired.
    ðŸ”—
    \((\Longleftarrow)\) Next, we assume \(g\ast h^{-1}\in H\) for all \(g,h\in H\) and we must show that \(H\) is a subgroup of \(G\text{.}\) To do this, we must show all of the axioms of a subgroup hold. In the hypotheses of the theorem, we were told that \(H\) is a nonempty subset of \(G\text{,}\) so that requirement is satisfied. For the second requirement, let \(h,k\in H\text{.}\) then by assumption we have \(k\ast k^{-1}\in H\text{.}\) As \(k\ast k^{-1}=e\text{,}\) we know \(e\in H\text{.}\) Now, we can see that our hypothesis gives \(k^{-1}=e\ast k^{-1}\in H\) and using the hypothesis one more time gives \(h\ast k=h\ast(k^{-1})^{-1}\in H\) as desired. For the final subgroup requirement, we must show that \(H\) satisfies the axioms of a group. This means showing that the operation on \(H\) is associative, has an identity in \(H\text{,}\) and every element has an inverse in \(H\text{.}\) These tasks are left for the exercises. For the identity part, looking carefully at the middle of this proof may give you what you need.

    ðŸ”—

    Remark 12.0.7.

    Often, the notation for the operation in a group is considered to be "multiplication" and the operation is written just as concatenation. With that standard, another way to phrase TheoremÂ 12.0.6 is to say that \(H\) is a subgroup of \(G\) if and only if for every \(g,h\in H\text{,}\) we have \(gh^{-1}\in H\text{.}\)

    12.C.

    Prove Lagrangeâ€™s Theorem for finite groups, which states that if \(G\) is a (finite) group, and \(H\) is a subgroup of \(G\text{,}\) then the order of \(H\) divides the order of \(G\text{.}\)

    Hint.
    One way to approach this involves definite cosets of a subgroup. To that end, let \(G\) be a group and \(H\) a subgroup of \(G\text{.}\) Define an equivalence relation, \(\sim\text{,}\) on \(G\) via \(x\sim y\) if and only if \(x^{-1}y\in H\text{.}\) We define the left cosets of \(H\) in \(G\) to be the equivalence classes of the equivalence relation \(\sim\) on \(G\text{.}\) With this language, the most common way to state Lagrangeâ€™s Theorem is that for any group \(G\) and any subgroup \(H\leq G\text{,}\) we have \(|G|=[G:H]\cdot|H|\) where \([G:H]\) is the index of \(H\) in \(G\), defined to be the number of left (or right) cosets of \(H\) in \(G\text{.}\)
